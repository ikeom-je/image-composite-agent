# プロダクト概要

画像合成REST API - AWSで構築された本番環境対応のサーバーレス画像処理システム

## 主要機能

- 1〜3枚の画像をアルファチャンネル対応で合成
- テキストテロップオーバーレイ（最大3テキスト、日本語対応、折り返し・背景矩形）
- 合成画像から動画を生成（MP4、WEBM、AVI形式）
- ドラッグ&ドロップによる直接S3画像アップロード
- リアルタイム画像プレビューとダウンロード
- HTTP URL、S3パス、テスト画像、アップロード画像に対応
- AIチャットエージェント（Strands Agents SDK + AWS Bedrock）
- マルチモデル対応（Claude Sonnet 4.5、Haiku等の切り替え）

## アーキテクチャ

- バックエンド: AWS Lambda (Python 3.12) + API Gateway
- フロントエンド: Vue.js 3 + Vite + Pinia
- ストレージ: S3バケット（リソース、テスト画像、アップロード、フロントエンド）
- CDN: 最適化されたキャッシュを持つCloudFrontディストリビューション
- インフラ: AWS CDK (TypeScript)

## 主要な特徴

- 高性能なARM64/X86_64 Lambda関数
- 並列画像取得（ThreadPoolExecutor）
- アルファチャンネル保持（RGBA）
- 動的設定管理
- 包括的な監視（CloudWatchダッシュボード、アラーム、ログ）
- セキュリティ: 署名付きURL、レート制限、CORS設定

## バージョン

現在: v3.2.0（マルチモデル対応、動画生成、テキストオーバーレイを含む）
