# Strands Agent チャットエージェント - ユースケース一覧

## 概要

Chat Agentの全ユースケースを体系化し、E2Eテスト・受入テスト・回帰テストの基盤とする。

## A. 画像合成

| # | ユースケース | 入力例 | 期待動作 | テストID |

## 未カバーのユースケース（今後のテスト拡充計画）

| # | ユースケース | 優先度 | 備考 |
|---|------------|--------|------|
| A4 | 座標位置指定 | 中 | LLM応答の座標値検証が困難 |
| A5 | サイズ指定 | 中 | 同上 |

| A9 | HTTP URL画像 | 低 | 外部依存 |
| A11 | 調整提案 | 低 | LLM応答の内容検証が非決定的 |
| B2 | WEBM指定 | 中 | B1のバリエーション |
| B4 | 一括指定 | 中 | 合成+動画の連続実行 |

| C2 | 画像削除 | 高 | テスト用画像のセットアップが必要 |
| C4 | 削除済み画像使用 | 低 | エラー応答の検証 |
| E1 | 複数ターン | 中 | LLMのコンテキスト保持の検証 |
| F4 | タイムアウト | 低 | 再現困難 |
