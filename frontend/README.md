# 画像合成REST API フロントエンド

このフロントエンドアプリケーションは、画像合成REST APIを使用して画像を合成するためのユーザーインターフェースを提供します。Vue.js 3で構築され、S3にホスティングされています。

## 機能

- 🎨 テスト画像を使用した画像合成
- 🔧 画像の位置とサイズのカスタマイズ
- 🌐 S3パスを使用した画像の指定
- 📱 レスポンシブデザイン
- 💾 画像のダウンロード機能
- 📋 API URLのコピー機能
- 📊 使用例の表示

## 開発環境のセットアップ

## S3へのデプロイとCloudFront配信

フロントエンドは自動的にCDKスタックの一部としてデプロイされます。S3バケットはプライベートに保たれ、CloudFrontを通じてのみアクセス可能です。

```bash
# プロジェクトのルートディレクトリで実行
cd ..
npm run deploy-all
```

このコマンドは以下の処理を行います：

1. フロントエンドアプリケーションをビルド
2. CDKスタックをデプロイ（S3バケットとCloudFrontディストリビューションの作成を含む）
3. ビルドしたファイルをS3バケットにアップロード
4. CloudFrontキャッシュを無効化

デプロイ後、CloudFormationの出力に表示される `FrontendUrl` にアクセスしてフロントエンドを確認できます。
このURLはCloudFrontのドメイン名を指し、HTTPS経由でセキュアにアクセスできます。

## 使用方法

1. フォームで画像合成のパラメータを設定
2. 「画像を生成」ボタンをクリック
3. 生成された画像を表示
4. 必要に応じて画像をダウンロードまたはAPI URLをコピー

## 使用例

アプリケーションには以下の使用例が含まれています：

1. **基本的な合成**: テスト画像を使用した基本的な合成例
2. **カスタム配置**: 画像の位置とサイズを調整した例
3. **S3画像の使用**: S3に保存された画像を使用した例

## API仕様

このフロントエンドは以下のAPIエンドポイントを使用します：

```
GET https://gv2g48xpz3.execute-api.ap-northeast-1.amazonaws.com/prod/images/composite
```

### 主なパラメータ

| パラメータ | 説明 | デフォルト |
|-----------|------|-----------|
| `baseImage` | ベース画像 | 透明背景 |
| `image1` | 合成する1つ目の画像 | テスト画像 |
| `image2` | 合成する2つ目の画像 | テスト画像 |
| `format` | 出力形式 (`html` or `png`) | `html` |
| `image1X` | 1つ目の画像のX座標 | 20 |
| `image1Y` | 1つ目の画像のY座標 | 20 |
| `image1Width` | 1つ目の画像の幅 | 300 |
| `image1Height` | 1つ目の画像の高さ | 200 |
| `image2X` | 2つ目の画像のX座標 | 20 |
| `image2Y` | 2つ目の画像のY座標 | 240 |
| `image2Width` | 2つ目の画像の幅 | 300 |
| `image2Height` | 2つ目の画像の高さ | 200 |

## トラブルシューティング

### CORS エラー

APIがCORSエラーを返す場合は、API GatewayのCORS設定を確認してください。

### S3アクセス権限エラー

S3バケットへのアクセス権限エラーが発生した場合は、バケットポリシーとブロックパブリックアクセス設定を確認してください。

### 画像が表示されない

画像が表示されない場合は、以下を確認してください：

1. APIエンドポイントが正しいか
2. テスト画像がS3バケットにアップロードされているか
3. Lambda関数がS3バケットにアクセスできるか
